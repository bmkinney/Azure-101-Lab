# Lab Agenda

## Recommended 120-minute agenda

### 0:00-0:10
Orientation
- lab goals
- sandbox assumptions
- tool locations in Azure portal

### 0:10-0:35
Build the core environment
- resource group
- VNet and subnets
- NSG and route table
- VM and storage account

### 0:35-0:50
VM troubleshooting
- identify platform, guest, or access issues

### 0:50-1:10
Network troubleshooting
- validate subnet, NSG, and route behavior

### 1:10-1:25
Azure Monitor and KQL triage
- review Activity Log
- run basic KQL queries

### 1:25-1:40
RBAC troubleshooting
- review role assignments and scope

### 1:40-1:55
Cost and policy review
- identify waste and governance constraints

### 1:55-2:00
Wrap-up
- recap lessons learned
- review teardown expectations
